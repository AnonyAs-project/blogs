# BlogSpace

A platform to write, and comment on blogs with ease.

BlogSpace allows users to write and share their own blogs, comment on others' posts, and engage with a community of writers to exchange ideas and thoughts.

## Installation Instructions

To get started with BlogSpace, follow these steps:

### 1. Clone the repository

```bash
git clone https://github.com/Mohamed-zakaria-coder/blogs.git

cd backend
npm install
cd ../frontend
npm install

```
### 2. Start the project

To start both the frontend and backend, run the following command in the root of the repository:

```bash
npm start
```
